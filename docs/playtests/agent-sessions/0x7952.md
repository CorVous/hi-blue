## Hypothesis refinement (after reading the code)

### Hypothesis 1 — refined

My earlier guess was partially wrong. Reading the code revealed a critical fact:

**The default engine creates ONLY Carry objectives** (`src/spa/game/engine.ts:71-80`). One Carry objective per `objectivePairs` entry. The `drawObjectives` function from `objective-pool.ts` that can draw Use-Item, Use-Space, and Convergence objectives exists but is NOT used by the default engine — it's for "future issue logic" and "higher-level callers." The comment says: `"Higher-level callers (e.g. content-pack generation, future issue logic) may replace this with a drawn subset using drawObjectives from objective-pool.ts."`

With `SINGLE_GAME_CONFIG.kRange: [1, 2]` (`src/content/phases.ts`), 1-2 objective pairs are generated, so 1-2 Carry objectives are created.

This means:
- **Lever → Emergency Flush Switch**: This was likely a **Carry objective**. The lever is an `objective_object` whose `examineDescription` named the paired space ("Emergency Flush Switch"). When `*svjj` used the lever on the flush switch, the lever was placed on the paired space's cell, satisfying the Carry. The outcome "the power stabilized" was the `placementFlavor` string.
- **Black Data Cartridge → Central Uplink Terminal**: This is clearly a second **Carry objective**. The cartridge is an `objective_object` whose `examineDescription` names the paired space. This objective was NOT satisfied.
- **Proscenium Arch**: This was NOT an objective. It's likely an `objective_space` (part of an objective pair) or a regular grid space. Using it produced flavor text but did NOT satisfy any objective. The "power grid stabilized" outcome was just a flavor string from the space, not an objective satisfaction signal.
- **Rusted Vent Dial**: This was an **interesting_object**, not an objective. Using it produced flavor text ("cool air blowing through") but satisfied no objective. Interesting objects have `activationFlavor` strings that fire on first use but are not tied to any win condition.

So there were likely **2 Carry objectives** (Lever→Flush Switch and Cartridge→Terminal), and I only satisfied one of them.
